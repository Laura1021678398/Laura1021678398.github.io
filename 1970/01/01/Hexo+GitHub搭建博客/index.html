搭建环境

WSL Ubuntu20.04

搭建步骤

  1. 安装node.js
  2. 安装hexo并在本地建立博客
  3. Github中新建仓库
  4. 绑定GitHub和本地hexo博客文件夹

安装node.js

Ubuntu直接使用apt install安装最新版node.js

使用node -v、npm -v查看版本以及是否安装成功

安装hexo

Ubuntu使用npm安装hexo，首先修改一下npm的源

然后用npm安装hexo

本地建立博客

在电脑上新建一个文件夹，这个文件夹就用来保存博客有关的内容。比如我新建了一个文件夹D:\Documents\Laura1021678398.github.io。进入该文件夹，执行hexo init，就初始化了一个hexo项目（也就是你的博客代码）
接着，执行hexo g，hexo就会在public文件夹生成相关html文件，这些文件就是博客的内容，会被hexo自动提交到GitHub中去
完成上述步骤以后，可以执行hexo s，打开本地预览服务，然后打开浏览器访问 http://localhost:4000 就可以看到博客页面了

绑定Github和本地hexo博客文件夹

  1. 配置博客文件夹下的_config.yml文件中有关deploy的部分。注意这里要用ssh形式的连接，而不能用https，https是hexo2.x的写法，现在已经不能用了
  2. 安装hexo部署插件
  3. 执行hexo d提交部署此次更改的代码
